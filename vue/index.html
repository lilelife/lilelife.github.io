vue

安装node.js

通过npm命令安装vuejs

在用 Vue.js 构建大型应用时推荐使用 NPM 安装，NPM 能很好地和诸如 Webpack 或Browserify 的 CommonJS 模块打包器配合使用。（以下操作全在命令行中）# 最新稳定版本
$ npm install -g vue 全局安装

最新稳定 CSP 兼容版本

$ npm install vue@csp
快速搭建大型单页应用

全局安装 vue-cli$ npm install -g vue-cli
创建一个基于 “webpack” 模板的新项目$ vue init webpack project-name
安装依赖$ cd project-name

$ npm install
$ npm run dev 启动项目
通过上面的一系列命令，我们便成功启动了项目，这时我们可以在浏览器中输入:
http://localhost:8080/
